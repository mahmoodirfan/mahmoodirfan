## Published Geospatial Tools

Tools published through the official [QGIS Plugin Repository](https://plugins.qgis.org) and [PyPI](https://pypi.org).

<table>
<tr>
<td width="50%">

**[RasterTrend](https://github.com/mahmoodirfan/RasterTrend)** &nbsp;`QGIS Plugin`

Pixel-wise Mann-Kendall trend analysis and Sen's Slope estimation on raster time-series stacks. Supports Standard and Seasonal Mann-Kendall tests.

`PyQGIS` `NumPy` `SciPy` `GDAL`

</td>
<td width="50%">

**TrendShift** &nbsp;`QGIS Plugin`

Pixel-wise abrupt change-point detection using Pettitt's non-parametric test. Outputs change magnitude, direction, p-value, and significance mask rasters.

`PyQGIS` `Rasterio` `SciPy` `NumPy`

</td>
</tr>
<tr>
<td width="50%">

**[spatialdrought](https://github.com/mahmoodirfan/spatialdrought)** &nbsp;`PyPI Package`

Pixel-wise spatial drought index computation on gridded remote sensing and climate datasets. Supports SPI, SPEI, VCI, TCI, VHI, and CDI.

`Python` `GeoPandas` `Rasterio` `NumPy`

</td>
<td width="50%">

**[OpenGeoEnrich](https://github.com/mahmoodirfan/OpenGeoEnrich)** &nbsp;`QGIS Plugin`

Automated geoenrichment integrating open global datasets, including population, land cover, elevation, and OSM infrastructure, for any user-defined study area.

`PyQGIS` `OSM Overpass API` `GeoPandas`

</td>
</tr>
</table>
